Network Architecture

Enterprise-style network setup with VLANs, switches, VPNs, and secure remote access.

Network Infrastructure

  • Switch: TP-Link 16-Port Gigabit Managed Switch with VLAN support.
  • VLANs:
    • VLAN 10 – Administration (Proxmox, TrueNAS, Plex server)
    • VLAN 20 – Gaming (Gaming PC, streaming PC, PS5)
    • VLAN 30 – General Access (IoT devices, guest network)
  • Remote Access & Security:
    • WireGuard VPN provides secure remote access with dynamic DNS for consistent connection.
    • Surfshark VPN on the Plex server encrypts external streaming traffic.
    • OPNsense VM acts as the firewall, managing all VLANs and network security rules.
  • Planned Enhancements: Reverse proxy with NGINX, centralized logging, intrusion detection systems, and enhanced firewall policies for enterprise-style monitoring.

Virtualization Layer (Proxmox Services)

Containerized and virtualized services providing modular, flexible, and secure lab operations.

Docker LXC (Alpine)

Role: Hosts a variety of containerized applications to manage network services, home automation, and self-hosted software.

Services Running:

  • Heimdall: Application dashboard
  • NGINX Proxy Manager: Reverse proxy and SSL management
  • Portainer: Docker management GUI
  • Uptime Kuma: Service uptime monitoring
  • Home Assistant: Home automation hub
  • Navidrome: Self-hosted music streaming
  • The Lounge: Web-based IRC client
  • Vikunja: Task management and project organization
  • Audiobookshelf: Audiobook streaming and library management
  • WordPress: Web server for content management and development testing

Purpose: Provides a modular, containerized environment for testing, deployment, and centralized control of multiple services without impacting core VMs.

OPNsense VM

Role: Enterprise-style firewall and routing solution.

Key Functions:

  • Manages all network traffic, VLAN segmentation, NAT rules, and port forwarding.
  • Provides VPN termination points and internal network monitoring.
  • Acts as the security gateway between external and internal networks.

Pi-hole LXC & WireGuard LXC

Pi-hole Role: Network-wide ad blocking and DNS filtering.

WireGuard Role: VPN server for secure remote connections.

  • Pi-hole improves network performance, reduces tracking, and centralizes DNS.
  • WireGuard provides secure remote access and integrates with VLAN routing for isolation.

Other VMs

  • Ubuntu VM: Linux testing environment for experimentation and scripting.
  • Windows Server 2025 VM: Testing Active Directory, group policies, and Windows services.
  • Windows 10 VM: Desktop environment testing for software deployment and Windows admin practice.

Storage, Backup, and Data Management

Ensuring redundancy, resilience, and quick recovery across all lab systems.
