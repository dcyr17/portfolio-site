Home Lab Infrastructure Overview

My home lab is a robust, self-hosted environment designed to simulate enterprise-grade IT infrastructure, providing a hands-on platform for virtualization, network management, systems administration, and secure remote access. It integrates multiple physical servers, virtual machines, containers, and Dockerized services, all orchestrated through Proxmox VE, providing a flexible, modular, and highly functional environment.

Physical Infrastructure

Proxmox Virtualization Server

Hardware: Intel Xeon E5-2660 v2 (10 cores / 20 threads), 68 GB ECC RAM, NVIDIA GTX 1050 GPU passthrough, SSD & HDD storage.

Role: Central virtualization host managing all core lab services, VMs, and containers.

Key Functions:

  • Hosts VMs and LXCs for network services, system testing, and containerized apps.
  • Provides GPU passthrough for accelerated workloads.
  • Supports VLAN segmentation and virtual bridges.
  • Snapshotting, backups, and isolated testing environments.

Plex Media Server

Hardware: Intel Xeon E5-1620 v2, 32 GB RAM, NVIDIA GTX 1060, running Windows.

Role: Dedicated media server optimized for streaming.

Key Functions:

  • Hosts personal media library for local & remote streaming.
  • Encrypted connections via Surfshark VPN.
  • Isolated VLAN for administrative and streaming purposes.
  • Backups to TrueNAS for redundancy.

TrueNAS Storage Server

Role: Centralized storage and backup with over 4TB capacity.

Key Functions:

  • Stores VM snapshots, container backups, and system images.
  • Provides NFS & SMB shares.
  • Monitors disk health via SMART and scheduled snapshots.

Gaming & Streaming PCs

Gaming PC: Ryzen 7 7800X3D, RTX 3060 Ti, 32 GB DDR5 RAM.

Streaming PC: Ryzen 5 5600X, RTX 3050, 32 GB DDR4 RAM.

Role: Isolated in gaming VLAN for high-performance workloads, media streaming, and network testing.

Network Architecture

Switch: TP-Link 16-Port Gigabit Managed Switch with VLAN support.

VLANs:

  • VLAN 10 – Administration (Proxmox, TrueNAS, Plex)
  • VLAN 20 – Gaming (Gaming PC, Streaming PC, PS5)
  • VLAN 30 – General Access (IoT devices, guest network)

Remote Access & Security:

  • WireGuard VPN for secure remote access with dynamic DNS.
  • Surfshark VPN on Plex for encrypted streaming traffic.
  • OPNsense VM manages firewall, VLANs, and security rules.

Planned Enhancements: Reverse proxy with NGINX, centralized logging, IDS, and advanced firewall policies.

Virtualization Layer (Proxmox Services)

Docker LXC (Alpine)

Role: Hosts containerized applications for network services, home automation, and self-hosted software.

Services Running:

  • Heimdall, NGINX Proxy Manager, Portainer, Uptime Kuma
  • Home Assistant, Navidrome, The Lounge, Vikunja
  • Audiobookshelf, WordPress

OPNsense VM

Role: Enterprise-style firewall and routing solution.

  • Manages all network traffic, VLAN segmentation, NAT rules, and port forwarding
  • Provides VPN termination points and internal monitoring
  • Acts as security gateway for perimeter defense

Pi-hole LXC (Alpine)

Role: Network-wide ad blocking and DNS filtering for performance and tracking reduction.

WireGuard LXC (Alpine)

Role: VPN server for secure remote access to home lab, integrated with VLAN routing.

Ubuntu VM

Role: Linux testing environment for experimentation, scripting, and learning.

Windows Server 2025 VM

Role: Enterprise testing for Active Directory, group policies, and Windows network scenarios.

Windows 10 VM

Role: Desktop testing for software deployment and Windows administration practice.

Storage, Backup & Data Management

TrueNAS handles all backups for Proxmox-hosted systems and containers (excluding large media). Snapshots, backups, and configuration files are scheduled regularly to ensure disaster recovery and minimal downtime. File integrity, SMART monitoring, and replication tasks provide enterprise-level resilience.
