Overview

My homelab runs Proxmox as the hypervisor. I use OPNSense for firewalling and VLANs, and Docker LXC containers for services (Heimdall, Nginx Proxy Manager, Portainer, Pi-Hole, Home Assistant, Plex).

Components

  • Proxmox VE — Virtualization host for VMs and LXC.
  • OPNSense — Firewall & VLAN management.
  • Docker LXC — Heimdall, Nginx Proxy Manager, Portainer, Uptime Kuma.
  • Pi-hole — DNS filtering and ad-blocking.
  • Plex — Media server on Windows VM.
