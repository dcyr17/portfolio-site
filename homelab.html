Home Lab Infrastructure Overview

My home lab is a robust, self-hosted environment designed to simulate enterprise-grade IT infrastructure...

Physical Infrastructure

  • Proxmox Virtualization Server: Intel Xeon E5-2660 v2, 68GB RAM, GTX 1050 GPU passthrough...
  • Plex Media Server: Windows, GTX 1060 GPU, Surfshark VPN...
  • TrueNAS Storage Server: 4TB capacity, NFS & SMB shares...

Network Architecture

  • VLAN 10 – Administration
  • VLAN 20 – Gaming
  • VLAN 30 – General Access

Virtualization Layer (Proxmox Services)

Docker LXC hosts services like Heimdall, NGINX Proxy Manager, Portainer, Uptime Kuma, Home Assistant, and more.

Security & Remote Access

WireGuard VPN provides secure remote access with OPNsense handling all firewall and VLAN management.
